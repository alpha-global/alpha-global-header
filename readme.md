# Alpha Global Header

* Stable tag: 1.0.6

### Overview

This package provides an Angular 4+ and a Polymer 1 global header element

### Angular

To develop:
```node
npm run angular:start
```

To build:
```node
npm run angular:build
```


### Polymer

To develop:
```node
npm run polymer:start
```

To build:
```node
npm run polymer:build
```

### Combined Build

To build all at once:
```node
npm run build
```
